# [pj-name] バックエンド構造テンプレート

このディレクトリは、[pj-name]（任意のプロジェクト名）のバックエンド構造テンプレートです。
流用時は[pj-name]を自分のプロジェクト名に置換してください。

## フォーマット例
- migrations/ : DBマイグレーション用SQL
- scripts/ : 各種スクリプト
- src/ : バックエンドソース
- src-client/ : フロントエンドソース
- tests/ : テストコード
- package.json : サンプル設定ファイル
